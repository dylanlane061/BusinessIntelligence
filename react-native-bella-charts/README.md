# react-native-bella-charts

Beautiful (bella) native charts for react native

## Installation

```sh
npm install react-native-bella-charts
```

## Usage

```js
import { BellaChartsView } from "react-native-bella-charts";

// ...

<BellaChartsView color="tomato" />
```
